# Word counter in lex

## Goal

Recreate the wc Unix command
./exe filename

Results:
File reads
- l: number of lines
- w: number of words
- c: number of caracs

## How to compile

Simply run
```
make
```

Compare with the example found on Github
```
make example
```

In order to clean, the useless file when you are finish:
```
make clean
```
